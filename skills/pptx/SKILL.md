# Skill: PPTX (PowerPoint)

Use esta skill quando o usuário enviar um arquivo `.pptx` ou mencionar apresentação, slides ou PowerPoint.

## Quando usar

- Usuário envia um arquivo `.pptx` no chat
- Usuário quer resumir, listar slides ou extrair texto de uma apresentação
- Usuário menciona PowerPoint, slides, deck, pitch ou apresentação

## Como funciona

Ao receber um `.pptx`, o bot oferece:
1. **Resumir** — Resume o tema e os pontos principais de cada slide
2. **Listar slides** — Lista todos os slides numerados
3. **Extrair texto** — Extrai o texto bruto de todos os slides

## Orientação para o usuário

Se o usuário mencionar uma apresentação mas não enviou ainda:
> "Pode enviar o arquivo .pptx aqui no chat que eu analiso os slides para você!"

## Exemplos de uso

- "Tenho um pitch deck, pode resumir os slides?"
- "Quantos slides tem essa apresentação?"
- "Extrai o texto desse PowerPoint"
