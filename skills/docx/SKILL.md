# Skill: DOCX (Word)

Use esta skill quando o usuário enviar um arquivo `.docx` ou mencionar que tem um documento Word.

## Quando usar

- Usuário envia um arquivo `.docx` no chat
- Usuário quer resumir, extrair ou analisar um documento Word
- Usuário menciona relatório, ata, comunicado, proposta ou qualquer documento de texto

## Como funciona

Ao receber um `.docx`, o bot oferece:
1. **Resumir** — Gera um resumo objetivo com pontos principais e próximos passos
2. **Extrair texto** — Retorna o texto bruto do documento

## Orientação para o usuário

Se o usuário mencionar um Word mas não enviou ainda:
> "Pode enviar o arquivo .docx diretamente aqui no chat!"

## Exemplos de uso

- "Tenho uma ata de reunião em Word, pode resumir?"
- "Extrai o texto desse documento Word"
- "O que diz essa proposta?"
