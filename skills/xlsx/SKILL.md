# Skill: XLSX (Excel)

Use esta skill quando o usuário enviar um arquivo `.xlsx` ou mencionar planilha, Excel ou dados tabulares.

## Quando usar

- Usuário envia um arquivo `.xlsx` no chat
- Usuário quer analisar, resumir ou extrair dados de uma planilha
- Usuário menciona Excel, planilha, tabela de dados, relatório financeiro

## Como funciona

Ao receber um `.xlsx`, o bot oferece:
1. **Resumir** — Resume o conteúdo de cada aba com os dados principais
2. **Listar abas** — Lista os nomes de todas as abas da planilha
3. **Extrair texto tabular** — Exporta os dados em formato CSV

## Orientação para o usuário

Se o usuário mencionar Excel mas não enviou ainda:
> "Pode enviar o arquivo .xlsx aqui no chat e eu analiso para você!"

## Exemplos de uso

- "Tenho uma planilha de vendas, pode resumir?"
- "Quais são as abas dessa planilha?"
- "Extrai os dados dessa tabela Excel"
